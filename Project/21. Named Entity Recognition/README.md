## 1. 实体识别的基本概念

### 1.1 实体识别的任务是识别出文本中三大类命名实体（实体类，时间类，数字类），具体如下：

- 实体类：人名、组织/机构、地理位置
- 时间类：时间、日期
- 数字类：货币、百分比
- ......
